# Glossary

> [!NOTE]
> The primary technical glossary of concepts in Engrama is currently maintained in Spanish, as it represents the core design vocabulary of the author team.
> An English translation of this glossary is planned and will be added in a future release.

To view the Spanish glossary with definitions of key terminology (such as *Entidades*, *Observaciones*, *Relaciones*, *Vault*, etc.), please switch the site language to **Español** using the language selector in the top-right corner of the header.

Alternatively, you can visit the Spanish version directly:
* **[Glosario de Términos (ES)](../../es/guides/glosario/)**
